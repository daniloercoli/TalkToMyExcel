Privacy

Static site, no marketing tracking.

This page describes the public website perimeter, separate from the TalkToMyExcel application used to process business datasets.

1. Controller and contact

For requests related to TalkToMyExcel, write to ttme@ercoliconsulting.eu.

2. Nature of the website

The public website is made of static HTML pages. It does not include external analytics, advertising pixels, web forms or profiling scripts.

3. Data collected

Data is collected only if the user voluntarily sends an email. In that case, the communicated data is used to reply to the request.

4. Demos and business data

Any files or business examples shared for a demo must be agreed separately. The public website does not receive uploads and does not directly process datasets.

5. Cookies

The public website does not set its own application cookies. Any technical logs from the hosting provider depend on the publication infrastructure.
