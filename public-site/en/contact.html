Guided demo

No need to sell magic. Show control.

In a useful demo, you inspect the structure and columns, ask a question and see which evidence produced the answer. The value is not only the summary: it is being able to verify it.

Use synthetic or anonymized data for the first conversation. If real data is needed later, we first agree on roles, provider, retention and a secure transfer channel: do not send it as an email attachment.
